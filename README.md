## How to Download

You can download the notebooks in a few ways:

### 1. Clone the Repository

If you have Git installed, you can clone the entire repository:

```bash
git clone https://github.com/airesearch-official/modal-notebook.git
```

### 2. Download a ZIP File

You can download a ZIP file of the entire repository by clicking on the 'Code' button on the GitHub repository page and selecting 'Download ZIP'.

### 3. Download Individual Notebooks

Navigate to the `notebooks` directory in the repository and click on the notebook you want to download. Then, click the 'Download' button.
