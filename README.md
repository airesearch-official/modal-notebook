## 📥 How to Use

### 1. Quick Download
You can download individual `.ipynb` files directly from the `notebooks/` folder and upload them to your [Modal.com](https://modal.com) dashboard.

### 2. Clone for Local Reference
```bash
git clone https://github.com/airesearch-official/modal-notebook.git
```

### 3. Running on Modal
1. Create a free account at [Modal.com](https://modal.com).
2. Upload any notebook from this repo to your Modal environment.
3. Select the recommended GPU (e.g., **A100** for LTX 2.3).
4. Run the cells and enjoy!
